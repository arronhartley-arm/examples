Debug and Trace Services Layer (DTSL)

Software examples for the Debugger's Debug and Trace Services Layer (DTSL). The examples are located in the archive file <examples_directory>\DTSL_examples.zip.

  • DTSL Example Configdb Example

    This directory contains the Debugger configdb entries used/referenced by the DTSL examples.

  • DTSL Java Example

    This is an example of how to use DTSL in 'stand-alone' mode (i.e. outside of the Eclipse environment). DTSL is mainly written in Java and so can be used by either a Java program, as in this example, or by a Jython (Python for Java) program.

  • DTSL Jython Example

    This is an example of how to use DTSL in 'stand-alone' mode (i.e. outside of the Eclipse environment). DTSL is mainly written in Java and so can be used by either a Java program or, as in this example, by a Jython (Python for Java) program.

  • DTSL Jython JTAG SVF Example

    This is an example of how to use DSTREAM to process Serial Vector Format (SVF) files and SVF statements. This uses DTSL to create a DSTREAM JTAG connection which is then used to interact with the JTAG SVF server on DSTREAM.

  • DTSL Jython Show Registers Example

    This example stand alone DTSL program will connect to a platform and show you the set of available registers for one device within the platform. What makes this example slightly more interesting than it first appears, is that because the example is simple it can attach to many platform types - including a models connection.

  • DTSL Probe Vcc Example

    This is an example of a stand alone DTSL program which connects to a DSTREAM or ULINK unit and shows its configuration items and their current values. Some of these configuration items allow the identification of the probe connection currently in use along with the state of target Vcc.

  • DTSL STM Decode Example

    This is an example of how to write your own STM processing scripts to decode the STM data streams, and to display Master IDs, Channel IDs and the data content.

  • DTSL STM File Decode Example

    This example shows how to decode STM trace data which has been captured into a file. The file data can be a DSTREAM/-ST/-PT/-HT buffer, an ETB buffer or just a raw STPv2 protocol stream.

  • DTSL Trace Dump Example

    This is an example of how you can dump a DTSL trace capture device's buffer to a disc file. The example has been written so that it may be used both as a stand alone DTSL program, or as a Debugger script.

  • DTSL Trace Stats Example

    This is an example of how one can process trace data using DTSL. This uses DTSL to create a trace processing pipeline which identifies all ATB ID sources within the trace capture device. This example can be run either stand alone or within a debug session.

  • ELA-500 Use-case Scripts Example

    This is an example of how one can use DTSL to interact with an Arm CoreSight ELA-500 Embedded Logic Analyzer. This uses DTSL and Use-case Scripts to configure and control the ELA-500. This example must be run within a debug session.

  • ELA-600 Use-case Scripts Example

    This is an example of how one can use DTSL to interact with an Arm CoreSight ELA-600 Embedded Logic Analyzer. This uses DTSL and Use-case Scripts to configure and control the ELA-600. This example must be run within a debug session.
